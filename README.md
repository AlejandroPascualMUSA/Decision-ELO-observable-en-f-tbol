# Decision-ELO-observable-en-f-tbol
La evaluación de eventos en fútbol describe lo que ocurrió. Este trabajo plantea una pregunta: dado el contexto en el momento de la acción, ¿el jugador eligió la mejor opción disponible? La inspiración procede del ajedrez: un motor no valora una jugada solo por su resultado inmediato, sino comparándola con la mejor jugada que podía realizarse.
